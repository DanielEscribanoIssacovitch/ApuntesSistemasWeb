Introducción a HTML

HTML (HyperText Markup Language) es un lenguaje de marcado utilizado para crear páginas web.

  • Hipertexto: texto con referencias a otros textos.
  • Creado por Tim Berners-Lee en 1991.
  • Estándar desde 1995 con HTML 3.0.
